# Kidney_GeoMx_Data_Analysis
This repository contains the codes demonstrating how to analyze Nanostring GeoMx Kidney DSP data.

The codes are adapted from the origanal vignette by Reeves et al at:
https://bioconductor.org/packages/release/workflows/vignettes/GeoMxWorkflows/inst/doc/GeomxTools_RNA-NGS_Analysis.html#41_Segment_QC
